# nc

## Description 
Where is the secret? <br>
Hello PwnWorld! <br>
nc.chall.cddc2020.nshc.sg 10000

## Solution 
Simply copy the command to a linux terminal to obtain the flag. 
```
root@kali:~# nc nc.chall.cddc2020.nshc.sg 10000
CDDC20{NiceToMeetYou@PwnWorld}
root@kali:~#
```

## Flag

CDDC20{NiceToMeetYou@PwnWorld}
